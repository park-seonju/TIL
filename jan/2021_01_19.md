- for -  else (break 이용)

  - ```python
    numbers = [1, 3, 7, 9]
    
    for i in numbers:
        if i == 4:
            print('True')
            break
    else:
        print('False')
        
    # False 출력  // 만약  numbers의 4가 있으면 True 하고 끝
    ```
